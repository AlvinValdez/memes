# Format details:

For meme generation, we currently have the following formats. It gives the textual representation of the memes after their creation.

### Format 1:

```
        __________________
        |  Text on top   |
        |                |
        |                |
        |                |
        |________________|
```

### Format 2:

```
        __________________
        |                |
        |                |
        |                |
        | Text in bottom |
        |________________|
```

### Format 3:

```
        __________________
        |  Text on top   |
        |                |
        |                |
        |                |
        |________________|
```

### Format 4:

```
        __________________________
        |    Text on the top     |
        |           |            |
        |           |            |
        |   Text in the bottom   |
        |___________|____________|
```

### Format 5:

```
Type 1: __________________________
        |     Long top text      |
        |           |            |
        |           |            |
        |    Text   |    Text    |
        |___________|____________|


Type 2: __________________________
        |   Text    |    Text    |
        |           |            |
        |           |            |
        |    Long bottom text    |
        |________________________|
```
